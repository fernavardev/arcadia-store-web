## Descripción

Arcadia Store es una aplicación web orientada a la presentación de juegos de mesa organizados por categorías, El proyecto incorpora navegación entre secciones, diseño responsive y un formulario de registro validado mediante JavaScript

## Tecnologías utilizadas

* HTML5
* CSS3
* Bootstrap 5
* JavaScript

## Funcionalidades

* Página principal con categorías de juegos
* Navegación entre categorías
* Diseño responsive para dispositivos móviles, tablet y escritorio
* Formulario de registro de usuarios
* Validaciones de formulario mediante JavaScript
* Manipulación dinámica del DOM
* Navegación entre categorías mediante enlaces anterior y siguiente
